# Сравнение скоростей структур данных

|          | Массивы                                                   | Связанные списки                              |
|----------|-----------------------------------------------------------|-----------------------------------------------|
| Чтение   | O(1)                                                      | O(n)                                          |
| Вставка  | O(n)<br/><small>(все двигаем вставляя в середину)</small> | O(1)<br/><small>(обновляем указатели)</small> |
| Удаление | O(n)<br/><small>(все двигаем удаляя из середины)</small>  | O(1)<br/><small>(обновляем указатели)</small>                                          |
