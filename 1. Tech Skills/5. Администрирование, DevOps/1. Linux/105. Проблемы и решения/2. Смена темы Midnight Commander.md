# Смена темы Midnight Commander
## Список тем
`/usr/share/mc/skins`

## Запустить с темой
`mc -S darkfar`<br/>

## Задать тему в конфигурации
`~/.config/mc/ini`<br/>
```sh
editor_filesize_threshold=64M
mcview_eof=
ignore_ftp_chattr_errors=true
skin=modarin256

[Layout]
message_visible=1
keybar_visible=1
```
